# Expanding Cards

A sleek, interactive image gallery where cards expand on click to reveal beautiful imagery and descriptive titles. This project is part of a larger collection of web components.

## ✨ Features

- **Interactive UI:** Smooth expansion animations using CSS transitions and cubic-bezier timing.
- **Responsive Design:** Automatically adjusts for mobile devices by hiding some cards to maintain a clean look.
- **Dynamic Styling:** Modern gradients and soft shadows for a polished feel.
- **Google Analytics:** Integrated with Google Tag Manager for tracking user engagement.

## 🚀 Getting Started

To run this project locally, simply clone the repository and open `index.html` in your favorite web browser.

```bash
# Clone the repository
git clone https://github.com/sergiogval/imagesUI.git

# Navigate to the project directory
cd imagesUI

# Open index.html (on Linux)
xdg-open index.html
```

## 🛠️ Technologies Used

- **HTML5:** Semantic structure.
- **CSS3:** Custom animations, Flexbox, and responsive media queries.
- **JavaScript (Vanilla):** DOM manipulation for the interactive expansion effect.
- **Google Fonts:** "Ubuntu" for clean typography.
- **Font Awesome:** Icon support.
